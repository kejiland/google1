# GitHub 节点过滤器

自动过滤并保留优质节点，移除低效协议。

## 🎯 过滤规则

**只移除以下开头的节点：**
- `http=`
- `https=`
- `socks5=`

**保留所有其他节点：**
- `ss://` 或 `ss=`
- `vmess://` 或 `vmess=`
- `vless://` 或 `vless=`
- `trojan://` 或 `trojan=`
- `ssr://` 或 `ssr=`
- 以及其他所有协议

## 🚀 自动执行

- **每天北京时间 8:00** 自动执行
- 结果保存到 `kejiland.txt`
- 自动提交到仓库
